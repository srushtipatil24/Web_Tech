data bindings means connect ui to code
| Type               | Symbol  | Direction | Use                 |
| ------------------ | ------- | --------- | ------------------- |
| Interpolation      | `{{ }}  | TS → HTML | Display data        |
| Property Binding   | `[ ]`   | TS → HTML | Set properties      |
| Event Binding      | `( )`   | HTML → TS | Handle events       |
| Two-Way Binding    | `[()]`  | TS ↔ HTML | Sync data both ways |
